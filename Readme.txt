The predictor pDHS-ELM is implement on the MATLAB platform on Windows  enviroment.To use it, please follow the following steps:
Step 1: Download the program and unzip it.
Step 2: please install the python program and modify the address of python in pDHS_ELM.m file to that address where the python program is install in your computer.
Step 3: open MATLAB, and run the pDHS-ELM in the command line as:
pDHS_ELM('test.fasta','test.out',0),then the predicting result can be found in the file 'test.out'
